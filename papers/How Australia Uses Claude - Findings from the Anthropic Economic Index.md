# How Australia Uses Claude: Findings from the Anthropic Economic Index

**来源**: https://www.anthropic.com/research/how-australia-uses-claude  
**发布时间**: 2026 年 3 月 31 日  
**作者**: Peter McCrory  
**数据来源**: 2026 年 2 月 Claude.ai 100 万条对话样本

---

## 核心结论

| 指标 | 数据 |
|------|------|
| 全球流量占比 | 1.6%，全球第 11 |
| 人均使用指数（AUI） | 4.1（是其劳动年龄人口预期值的 4 倍以上） |
| 全球人均排名 | 第 7（仅次于新加坡、以色列、卢森堡、瑞士、美国、加拿大） |
| 工作用途 | 46% |
| 学习用途 | 7% |
| 个人用途 | 47% |
| AI 自主度评分 | 3.38 / 5（偏低，更倾向协作而非委托） |

---

## 州级分布

澳大利亚的使用高度集中在两个州：

| 州 / 领地 | 对话占比 | AUI |
|-----------|---------|-----|
| 新南威尔士州 | 37.2% | 1.20 |
| 维多利亚州 | 30.8% | 1.19 |
| 昆士兰州 | 17.7% | < 1 |
| 西澳大利亚州 | 7.6% | 0.68（最低之一） |
| 南澳大利亚州 | 4.6% | < 1 |
| 首都领地 | 1.4% | < 1 |
| 塔斯马尼亚州 | 0.6% | 0.32 |
| 北领地 | 0.1% | 0.12（最低） |

**规律**：只有新南威尔士和维多利亚的 AUI > 1，其余州均低于预期。

### 为什么收入不能预测各州采用率？

与跨国比较（越富裕的国家 AUI 越高）不同，澳大利亚州级数据中**收入与 AUI 不相关**：

- 西澳大利亚州（GSP 最高，矿业主导）→ AUI 最低之一
- 首都领地（收入高于平均，大量公共部门雇员）→ 采用率低于预期
- 新南威尔士 / 维多利亚（收入略低于均值）→ 采用率最高

解释：驱动因素是**劳动力结构**（金融、专业服务、科技行业占比），而非人均收入。

## 与英语圈同类国家的对比

澳大利亚整体与英语圈国家（美、英、加、新西兰、爱尔兰）高度相似：

- 工作占比（46%）处于中位
- 个人用途（47%）接近最高
- 学习用途（7%）偏低——高收入高采用率国家的普遍特征
- AI 自主度（3.38）偏低——更倾向协作式使用

### 任务复杂度的有趣组合

| 维度 | 澳大利亚 | 全球均值 |
|------|---------|---------|
| 提示词理解所需教育年限 | **11.9 年**（高于中位） | 低于 11.9 |
| 无 AI 完成该任务预估耗时 | **2.7 小时** | 3.3 小时 |

结论：澳大利亚用户的提示词教育水平高，但任务本身更短小。

---

## 澳大利亚最显著的特征：任务多样性

澳大利亚的 Top 100 任务仅覆盖其 **47.3%** 的使用量，低于：
- 美国（47.7%）
- 英国（48.3%）
- 加拿大（50.2%）
- 全球均值（52.3%）

→ 澳大利亚是英语圈中**使用场景最分散**的国家。

### 任务构成偏差（vs 全球均值）

**低于全球的类别：**

| 类别 | 偏差 |
|------|------|
| 计算机与数学（编程） | **−8.0pp**（最大缺口） |
| 教育相关 | −2.7pp |
| 文档翻译 | 偏低（英语母语市场，翻译需求少） |

**高于全球的类别：**

| 类别 | 偏差 |
|------|------|
| 管理（Management） | +2.3pp |
| 办公与行政支持 | +1.3pp |
| 自然与社会科学 | +1.3pp |
| 个人生活管理 | +1.9pp |
| 健康与健康支持 | +1.8pp |
| 职场通讯 | +1.7pp |
| 商业文档 | +1.6pp |
| 财务指导 | +1.3pp |

### 与英语圈同类的对比

虽然编程低于全球是所有英语圈国家的共同特征（英语圈均值 −8.9pp），但**填补编程缺口的方向**不同：

- 英语圈整体：教育相关 +1.6pp 填补
- 澳大利亚：管理 +2.3pp、行政支持 +1.3pp 填补，教育反而更低（−2.7pp）

## 方法论说明

- 数据来自 2026 年 2 月抽取的 100 万条 Claude.ai 对话
- 任务分类采用 O*NET（美国劳工部职业任务分类体系）
- AUI（Anthropic AI Usage Index）= 实际使用量 / 劳动年龄人口预测值，1.0 为与人口规模持平

## 个人理解

这篇文章本质上是 Anthropic Economic Index 系列的国家/地区分析版本，核心价值在于：

1. **验证了结构性规律**：人均收入跨国预测 AI 使用，但在国家内部是劳动力结构（而非收入）在起作用
2. **澳大利亚的独特性**：不是"更多编程"，而是"更广泛的非技术专业用途"——这反映了经济结构而非技术偏好
3. **对 Claude 产品的含义**：高采用率经济体更倾向低自主度的协作模式，不是"把任务全扔给 AI"，而是"和 AI 一起做"
